  • Reserve subsystem name from Nokia Registry System
  • Fill out Flexman's VE request form
  • Wait for email response from flexman.

Viewing source code

SVN

iSource project home page

https://isource.nokia.com/projects/SS_TestXXX

ViewVC

ViewVC access for browsing code changes is available individually for each subsystem in e.g. https://isource.nokia.com/plugins/scmsvn/viewvc.php/?root=SS_TestXXX

Direct diff between two tags

For example: 

svn diff \
https://svne1.isource.nokia.com/isource/svnroot/SS_TargetInfo/tags/SS_TESTXXX_3.1 \
https://svne1.isource.nokia.com/isource/svnroot/SS_TargetInfo/tags/SS_TESTXXX_3.2

For graphical diff use kompare: 

svn diff ... | kompare -o -

Clearcase

View VE history 

cleartool lshist <filename>

Direct diff between two tags

To diff two versions: 

cleartool diff XercesHelper.cpp@@/main/int_flexiserver2_hki2/2 XercesHelper.cpp@@/main/int_flexiserver2_hki2/1

To diff file to previous version: 

cleartool diff -predecessor XercesHelper.cpp

Xcleardiff can be used for graphical diffs: 

xcleardiff XercesHelper.cpp@@/main/int_flexiserver2_hki2/2 XercesHelper.cpp@@/main/int_flexiserver2_hki2/1

Vediff

vediff is an useful scm script for diffing published labels (= versioned entities). 

cd
mkdir diff
cd diff
vediff SS_TestUTIL_2.14 SS_TestUTIL_2.15

Or with kompare: 

vediff -c SS_TestUTIL_2.14 SS_TestUTIL_2.15
kompare SS_TestUTIL_2.14 SS_TestUTIL_2.15
