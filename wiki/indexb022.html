[edit] Basic Formatting

To edit existing page click the corresponding [edit] link on the right for editing sections separatelly or at top to edit whole page as single file.

To create new page add link to it and click the corresponding link: 

[[Integration_Project_FP5_Cat_BI_Wiki_Usage]]

Normal text can just be typed. Single line brakes are ignored. This applies also to links, so add empty line between links if you want to have them at seperate lines.

So 

Normal text can just be typed.
Single line brakes are ignored.

Produces:
Normal text can just be typed. Single line brakes are ignored.

Empty line or <br> acts as line break.

Use '''bold''' (three single quotes on each side) for bold text.

Use ''Italic'' (two single quotes on each side) for Italic text.

Single quotes produce 'Single quotes'

Quotes can be combined, so ''''Bold quote'''' (four single quotes on each side) shows up as 'bold' text, five quotes as Bold Italic and six quotes as 'Single quote Bold Italic'.

Horizontal line (divider) can be made with <hr> or ----

text can be centered with <center>text</center>.

Wiki formatting can be ignored with <nowiki>Some text</nowiki>. These can be cascaded to print <nowiki></nowiki> respectedly.

Code and preformatted monospace text can be embedded with space as first characted on each line (other spaces are used as indentation): 

public class Hello {
   public static void main(String[] args) {
      System.out.println("Hello world!");
   }
}

Use "* item1" for non-numbered list and "** subitem" for subitems:

  • item1
    • subitem
  • item2

Use "# elem1" for numbered list and "## subelem" for subelems:

  1. elem1
    1. subelem
  2. elem2

Tabs can be made with : character as first character(s) in line:

No tab

1 tab=:
2 tabs=::
3 tabs=:::
...

[edit] Colors

Other ways of emphasizing text should be used in favor of colors: 

<strong>strong</strong>

strong 

<em>em</em>

em

If you really want to use colors here's the way: 

<span style="color:green">green</span>

Which produces:

green

style element can be applied to some other html tags also, see MediaWiki Help:Tags for more information.

[edit] Sections

You can add heading and sections with = characters: 

= Level 1 headline =
== Level 2 headline ==
=== Level 3 headline ===
= Another level 1 headline =

which produces:
