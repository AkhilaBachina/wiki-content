[edit] Regression suites

The regression suites we use in FP5-Cat are FlexiTAF jobplans and use various metadata of the automated testcases (see concept here ).
The jobplans are part of the testcase repository: http://sugar.fp.nsn-rdnet.net/cgi-bin/viewvc.cgi/rel_5.cat/FunctionalTesting/common/jobplans/

Detailed descriptions of the jobplans can be found here:

  • BI and BI-related jobplans
  • Regression suite execution jobplans
  • Common-use jobplans

How to set the correct testcase metadata is described here (wiki instructions).
