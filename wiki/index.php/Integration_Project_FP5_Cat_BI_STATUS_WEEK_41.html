CAT MIPS 

Last Working Build: R_FSPR5CAT_1.11.WR.mips.0810101011.2135

- All services seems to OK. Except CLicMgr. Pronto 27503ESPE01 Exists.

CAT Virtual Environment 

Last working build: R_FSPR5CAT_1.11.WR.32.0810082030.2130

- Tried with ECOS deployment. All services seems to be OK.

- ConfMgmtActivator and ConfMgmtValidator are NOK. Fixed in latest build.

- Directory comes up after 10min. Fix coming in the latest build.


CAT ATCA 

Last working build: R_FSPR5CAT_1.11.WR.32.0810101249.2135

- You have to try with "create_2nodes_nsnatca_AHUB2.sh" deployment instead of create_nsnatca.sh. Because create_nsnatca.sh deployement is not compatible with BPBOOT in the build. Fix coming in the next build.

- Directory comes up after 10min. pronto: 27448ESPE01. Fix coming in the next build.

- SSH doesn't work because of RoutingDaemon issue. With following work around you can make SSH working.

1. First lock /RoutingDaemon RG
2. add default route as follows.
ex:
ip -4 route add default via 10.20.100.1 dev eth4 proto static
3. Restart SSH daemon as follows
hascli -rn /CLA-0/FSDirectoryServer/SSHDaemon

- Tomcat doesn't come up. Fix coming in the next build.

- AS nodes doesn't boot (Implementation still ongoing).
