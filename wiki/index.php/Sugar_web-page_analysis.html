Restructuring/Redesigning ideas:

On some pages, boxes are using different size within the same page, it might be nice to keep the same size. Also some pages use the different colorscheme and/or fonts. Some files are simple html page, they may be also created with automatic page generation methods. Some extra features might be added, for example if any possibility to connect to actual machine exists, the link might appear, clicking on which the user will be able to login to particular machine.
As for design, in my opinion, the main page should be redesigned. Menues might be used, e.g. we can create a menu called “AutomationStatus” and the submenus would be “Today’s executions”, “AutomationFrame sessions ……”, etc. (The idea is to keep the same structure of links, but reorganize the view, so that it will be better readable). Of cause, no complications are needed, simple menus will be more than enough.
Also, there is a discussion amongst psycologists, that it is better to use icons in interfaces. So, we may also utilize this idea.
