[edit] Servers

[edit] Hosts

  • esling{31,32,33,39}.emea.nsn-net.net reserved exclusively for Component CI use.
  • esfleg38.emea.nsn-net.net is used to run unit test projects.

Masters and UIs running in:

  • Lynx, Cat: http://esling31.emea.nsn-net.net:8080/
  • Cougar: http://esling32.emea.nsn-net.net:8080/
  • Use "admin" account to configure servers

[edit] Tomcat instances

  • Tomcat servers running in /var/fpwork/hudson/servers in each server.
  • startup and shutdown with bin/startup.sh/shutdown.sh.
    • There are some local modifications in startup.sh

[edit] Scripts and configuration

Running config in /var/fpwork/hudson/configuration/server (local copy) repository: https://svne1.access.nokiasiemensnetworks.com/isource/svnroot/scm_fp/flexman/ci/trunk/subsystems/build/

The official workspace is used only to check SCM changes. No builds there. See actual compilation host and workspace address in console output.

[edit] Accounts

All servers are running with fpciman account. The account has shared NFS home directory between LinSEE hosts.

[edit] Updating Hudsons

  • Use pwget to download new WAR in ~/hudsonWAR/version/
  • cp hudson.war /var/fpwork/hudson/servers/server/webapps/ROOT.war
  • restart
  • Don't upgrade Doxygen-plugin

[edit] Email notifications

Email notifications require the correct mail server is configured in System Config. The easiest way to temprarily disable email notifications is to remove the mail server name from the config. The required values are in "LDAP Email Lookup": 

server: ldap://vd.emea.nsn-net.net:10391
BaseDN: ou=People, o=Nokia
BindDN:
BindPassword:
Email Attribute: mail
Search Attribute: uid
Perform Search: check

and in "E-mail Notification": 

SMTP server: demuexc024.nsn-intra.net
Default user e-mail suffix: 
System Admin E-mail Address: Hudson@esling32
Hudson URL: http://esling32.emea.nsn-net.net:8080/

[edit] Groovy Scripts

[edit] Enable all jobs 

import hudson.model.*
for(item in Hudson.instance.items) {
  println("JOB : "+item.name)
  item.disabled=false
  item.save()
}

[edit] Start all jobs 

import hudson.plugins.emailext.*
import hudson.model.*
import hudson.maven.*
import hudson.maven.reporters.*
import hudson.tasks.*
joblist = hudson.model.Hudson.instance.items.findAll{job -> job.isBuildable()}   
startServer = "admin computer"
startNote   = "bulk start"
cause = new hudson.model.Cause.RemoteCause(startServer, startNote)
joblist.each{run -> run.scheduleBuild(cause)}

[edit] Restart aborted jobs 

import hudson.plugins.emailext.*
import hudson.model.*
import hudson.maven.*
import hudson.maven.reporters.*
import hudson.tasks.*
activeJobs = hudson.model.Hudson.instance.items.findAll{job -> job.isBuildable()}
failedRuns = activeJobs.findAll{job -> job.lastBuild != null && job.lastBuild.result == hudson.model.Result.ABORTED}
startServer = "admin computer"
startNote   = "bulk start"
cause = new hudson.model.Cause.RemoteCause(startServer, startNote)
failedRuns.each{run -> run.scheduleBuild(cause)}

[edit] Empty build queue 

import hudson.plugins.emailext.*
import hudson.model.*
import hudson.maven.*
import hudson.maven.reporters.*
import hudson.tasks.*
for(item in Hudson.instance.queue.items) {
   println("Cancel build: $item");
   Hudson.instance.queue.cancel(item);
}
