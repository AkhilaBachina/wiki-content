[edit] RunCommand()

Name:

  • RunCommand

Description:

  • Runs a given command and checks the outcome

Usage:

  • RunCommand [options]

Created:

  • 10.02.2006 Antero Nousiainen

Version:

  • 0.5

Last Modified:

  • 25.04.2006 Antero Nousiainen

Version history:

  • 0.1 First version
  • 0.2 Added new option: -n, --negative
  • 0.3 Added new option: -q, --quiet
  • 0.4 Removed formatting from the command output
  • 0.5 Modified to be independent of other functions

Parameters:

  • -c, --command [command] The command that is to be executed
  • -e, --error [error msg] The error message that is shown should the given command fail
  • -i, --indent [indent level] Indent level
  • -n, --nonzero Exit state should be non-zero
  • -o, --operation [command description] Short description of the upcoming operation
  • -q, --quiet Command output message is not echoed on screen

Dependecies:

  • None

Returns:

  • 0 if the operation succeeded
  • 1 if the command exit state was not what was expected
  • 2 if incorrect number of parameters given
  • 3 if incorrect option given
  • Command return message is store in REPLY variable

E.g. 

RunCommand \
    --operation Show current date and time \
    --command date \
    --error Failed to show current date and time \
    --indent 1

Output: 

Show current date and time:
    Time stamp: 09:39:16
    Command   : date
    Exit state: 0
    Expected  : 0
    Result    : OK
    Message   : Tue Apr 25 09:39:16 EEST 2006
