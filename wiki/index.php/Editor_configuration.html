[edit] XEmacs

[edit] Configuration files

Main configuration file is ~/.xemacs/init.el

Common configuration file for all common inits is /twafs/utilities/emacs/common-init.el.

We have some ready made configurations in /twafs/utilities/emacs for flexiaf environments (not LinSEE). Those can be imported with require command: 

;; Load TFRM definitions
(setq load-path (cons (expand-file-name "/twafs/utilities/emacs") load-path))
(require 'common-init)
(require 'flexitaf-mode)
(require 'goto-matching-paren-or-insert)

[edit] Tabs

Add this command to the configuration file to force editor to insert spaces instead of tab chars 

;; Never ever insert tab characters
(setq-default indent-tabs-mode nil)

[edit] Remove tabs from an existing file

Use this command sequence 

M-x mark-whole-buffer (also C-x h)
M-x untabify

[edit] Indentation

[edit] Setting style and offset 

(setq c-basic-offset 4)
(setq c-default-style "stroustrup")

See also /twafs/utilities/emacs/flexitaf-mode.el, and preferably use it.

[edit] Bind F12 to indent whole buffer 

(defalias 'indent-all
  (read-kbd-macro "C-x h ESC C-\\"))
(global-set-key 'f12 'indent-all)

[edit] Indent whole buffer

Press F12 if you have defined the binding above, or use this command sequence 

M-x mark-whole-buffer (also C-x h)
M-x indent-region (also M-C-\)

[edit] Syntax highlighting 

(load-library "font-lock")
(setq global-font-lock-mode 1)

[edit] Use C++ mode for .h files 

(setq auto-mode-alist (cons '("\\.h$" . c++-mode) auto-mode-alist))

[edit] Enable mouse wheel 

(mwheel-install)

[edit] Display cursor position 

(setq line-number-mode 1)
(setq column-number-mode 1)

[edit] Compilation

[edit] Default compile command 

(setq compile-command "clearmake -C gnu")

[edit] Bind F4, F5, F6, F7 to compile and goto errors 

(global-set-key [f4] 'compile)
(global-set-key [f5] 'first-error)
(global-set-key [f6] 'next-error)
(global-set-key [f7] 'previous-error)

[edit] Maven 2

There are some helpers in "/twafs/utilities/emacs/mvn-compile.el"

[edit] Vim

[edit] Configuration files

Main configuration file is ~/.vimrc

[edit] Tabs

Add this command to the configuration file to force editor to insert spaces instead of tab chars 

set expandtab

To set tab offset to 4 

set tabstop=4
set shiftwidth=4

[edit] Remove tabs from an existing file 

:retab

[edit] Indentation

To enable indentation suitable for C-style syntax 

set autoindent
set smartindent

[edit] Disable indentation when copy-pasting text

Don't put this in the configuration file, but this command can be used to temporarily disable indentation and some other features if pasting text in terminal to vim. 

set paste

Return to normal 

set nopaste

[edit] Syntax highlighting 

syntax on

[edit] Display cursor position

Displays the cursor position in the lower right corner: 

set ruler

Displays a line number in front of every line (Not that useful. Makes copying difficult): 

set number

[edit] Display mode

Shows the editor mode: insert/visual etc. 

set showmode

[edit] Emacs

For the most part Emacs and XEmacs are the same product and XEmacs configuration instructions can be applied to Emacs. Main configuration file is ~/.emacs (corresponds XEmacs' ~/.xemacs/init.el)


[edit] Vi

Vim is Vi Improved. Try to use it instead. On most systems vi command actually starts vim. Main configuration file is ~/.exrc
